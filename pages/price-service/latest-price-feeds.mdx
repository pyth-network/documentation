# Latest Price Feeds

**/api/latest_price_feeds**

Get the latest price update for a given set of price ids.

| Argument                                  |                             | Description                                                                                                                                |
| ----------------------------------------- | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| <Arg required={true} type="hex">ids</Arg> | <Input id="ids[]" />        | The ID of the price update to retrieve.                                                                                                    |
| <Arg type="boolean">verbose</Arg>         | <Input id="verbose" />      | If true, return extra metadata in the response.                                                                                            |
| <Arg type="boolean">binary</Arg>          | <Input id="binary" />       | If true, return the binary price update data in the response. (deprecated, use `target_chain` instead)                                     |
| <Arg type="enum">target_chain</Arg>       | <Input id="target_chain" /> | If provided, get the binary price update data formatted in the proper string format for the given chain. Choose from `["evm", "default"]`. |

Examples:

<Example
  keyValues={{
    "ids[]": (ctx) => ctx.getFeedId("Crypto.BTC/USD"),
    target_chain: () => "evm",
  }}
>
  BTC/USD on EVM
</Example>

<RestGet
  endpoint="https://xc-mainnet.pyth.network/api/latest_price_feeds"
  queryParams={["ids[]", "verbose", "binary", "target_chain"]}
/>
