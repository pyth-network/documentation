# Asset Classes

Pyth [price feeds](https://www.pyth.network/price-feeds) provide data for the following asset classes:

| Asset Class | Subclass         | Definition |
| ----------- | ---------------- | ---------- |
| Crypto      | Spot Prices      |            |
|             | Redemption Rates |            |
|             | Indices          |            |
| US Equities | Spot Prices      |            |
| FX          | Spot Prices      |            |
| Metals      | Spot Prices      |            |
| Rates       | Future Prices    |            |
| Commodities | Futures Prices   |            |
