# Get Stale Price Threshold Secs

Get the default stale price threshold in seconds.
This quantity is the maximum age of price updates returned by functions like [get_price](get-price) and [get_ema_price](get-ema-price);
these functions revert if the current on-chain price is older than this threshold.

## Example Code

<Tabs items={['Move']}>
    <Tab>
        <DynamicCode targets={{}}>

            ```rust copy
            use pyth::price_identifier;

            pyth::get_stale_price_threshold_secs();
            ```
