# Entropy on EVM

## SDK

The [Solidity SDK](https://github.com/pyth-network/pyth-crosschain/tree/main/target_chains/ethereum/entropy_sdk/solidity#pyth-entropy-solidity-sdk) enables developers to easily integrate Pyth Entropy into EVM applications.
Please see the README there for a simple integration guide.
