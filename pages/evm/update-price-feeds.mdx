# Update Price Feeds
**updatePriceFeeds(updateData: bytes[])**

Update the on-chain price feeds using the provided data payloads.
You can retrieve these data payloads using the Price Service API.

| Argument                                                  |                           | Description |
|-----------------------------------------------------------|---------------------------| --- |
| <Arg required={true} type="hex[]">updateData</Arg>        | <Input id="updateData" /> | The price update data for the contract to verify. |
| <Arg required={true} type="wei">fee</Arg>                    | <Input id="fee" />           | The update fee in wei. |
